What It Does

Core Features

Club Organizer Weekend Player Stats Analyst
🏏

Match Setup

Team names, overs, toss & batting choice — configured in seconds before the first ball.

⚡

Ball-by-Ball Scoring

12 event types: 0–6 runs, Wicket, Wide, No Ball, Bye, Leg Bye. Undo last ball always available.

📊

Live Dashboard

Score, run rate, target, RRR, current batsmen and bowler — updated live after every ball.

👤

Player Statistics

Batting: runs, SR, 4s, 6s. Bowling: overs, wickets, economy. Full figures for every player.

📈

Team Analytics

Runs, wickets, extras, boundaries count, and run rate graph visualised over the full innings.

🏆

Match Summary

Winner, margin, scorecard, Man of the Match — share as image or PDF in a single tap.
